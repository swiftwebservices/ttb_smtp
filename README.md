# SMTP Portal Tester Sample

This is a simple Python script that tests SMTP server connectivity and email delivery. It's ideal for developers verifying SMTP setups before purchasing or integrating.

## Requirements
- Python 3.x
- No additional libraries needed (uses built-in smtplib)

## Usage
1. Replace placeholders with your SMTP details.
2. Run the script: `python script.py`

The script attempts to send a test email and reports success or failure.
